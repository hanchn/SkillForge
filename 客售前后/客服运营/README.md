# 客服运营

## 分类定位

管理跨渠道、多语言售前售中售后接待、话术、权限、升级与客户声音。

## 选择原则

- 先确认国家、渠道、语言、商品与订单事实源、服务阶段和坐席权限。
- 客售前后负责购买前咨询、订单中服务、售后处理、知识和 VOC；渠道运营负责店铺经营规则，市场采购与仓储库存提供商品、供应和库存履约事实。
- 不得为转化或平息投诉虚构功能、现货、折扣、送达、退款或赔付承诺；敏感语言和高风险事项必须人工复核。

## 当前 Skill

- [跨境客售前后运营专家](customer-service-voice-manager/README.md)：端到端完成购买前咨询、商品推荐、订单中服务、售后处理、多语言沟通、知识沉淀、质检培训和跨部门交接。适用于邮件、聊天和平台消息的售前售中售后接待、商品推荐、订单查询、异议、退款退货与多语言沟通、客服知识库、投诉升级、线索交接和质量改进，交付客售前后答复与处理方案、客服 SOP、FAQ、话术、权限和升级矩阵、咨询、售后、质检、培训与 VOC 改进报告。
