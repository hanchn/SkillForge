# 客售前后资深经理

## 分类定位

客售前后资深经理主角色、责任边界和配套周报 Skill。

## 选择原则

- 先确认国家、渠道、语言、商品与订单事实源、服务阶段和坐席权限。
- 客售前后负责购买前咨询、订单中服务、售后处理、知识和 VOC；渠道运营负责店铺经营规则，市场采购与仓储库存提供商品、供应和库存履约事实。
- 不得为转化或平息投诉虚构功能、现货、折扣、送达、退款或赔付承诺；敏感语言和高风险事项必须人工复核。

## 当前 Skill

- [客售前后资深经理](senior-customer-service-presales-manager/README.md)：统筹跨渠道售前咨询、订单中服务、售后处理、商品推荐、多语言沟通、知识库、质检培训和跨部门交接。适用于由客售前后资深经理接管复杂经营任务、制定季度或月度计划、跨专项协同、经营诊断和复盘，交付角色经营目标与责任边界、专项 Skill 编排与决策、经营节奏、风险和团队行动计划。
- [客售前后资深经理周报](senior-customer-service-presales-manager-weekly-report/README.md)：基于真实经营证据生成客售前后资深经理视角的结论先行周报，并跟踪目标、驱动、风险、决策和责任闭环。适用于客售前后资深经理本周经营复盘、本周与上周或目标对比、管理层同步和跨团队行动跟踪，交付角色经营周报、指标变化和驱动解释、风险、决策与下周行动表。
