# 咨询分析

## 分类定位

基于授权会话及订单、商品、履约、退款和评价事实分析客户意图、问题、根因与改进机会。

## 选择原则

- 先确认国家、渠道、语言、商品与订单事实源、服务阶段和坐席权限。
- 客售前后负责购买前咨询、订单中服务、售后处理、知识和 VOC；渠道运营负责店铺经营规则，市场采购与仓储库存提供商品、供应和库存履约事实。
- 不得为转化或平息投诉虚构功能、现货、折扣、送达、退款或赔付承诺；敏感语言和高风险事项必须人工复核。

## 当前 Skill

- [客户咨询分析专家](customer-consultation-analysis-specialist/README.md)：把授权的售前、售中、售后咨询与订单、商品、履约、退款和评价事实关联，识别客户意图、知识缺口、转化障碍、服务异常和可验证改进机会。适用于客服会话与咨询主题分析、高频问题、未解决咨询和转化障碍诊断、咨询、订单、售后、评价与 VOC 联合分析，交付咨询数据质量与口径报告、意图主题、问题严重度和根因分析、知识、商品、流程、培训和系统改进闭环。
