# 客售前后

## 分类定位

覆盖购买前咨询、订单中服务、售后处理、多语言沟通、知识库、咨询分析、质检培训与跨部门交接。参见 [客售前后 Skill 地图](CUSTOMER_SERVICE_SKILLS_MAP.md)。

## 选择原则

- 先确认国家、渠道、语言、商品与订单事实源、服务阶段和坐席权限。
- 客售前后负责购买前咨询、订单中服务、售后处理、知识和 VOC；渠道运营负责店铺经营规则，市场采购与仓储库存提供商品、供应和库存履约事实。
- 不得为转化或平息投诉虚构功能、现货、折扣、送达、退款或赔付承诺；敏感语言和高风险事项必须人工复核。

## 当前 Skill

- [客户咨询分析专家](咨询分析/customer-consultation-analysis-specialist/README.md)：把授权的售前、售中、售后咨询与订单、商品、履约、退款和评价事实关联，识别客户意图、知识缺口、转化障碍、服务异常和可验证改进机会。适用于客服会话与咨询主题分析、高频问题、未解决咨询和转化障碍诊断、咨询、订单、售后、评价与 VOC 联合分析，交付咨询数据质量与口径报告、意图主题、问题严重度和根因分析、知识、商品、流程、培训和系统改进闭环。
- [客售前后资深经理](客服角色/客售前后资深经理/senior-customer-service-presales-manager/README.md)：统筹跨渠道售前咨询、订单中服务、售后处理、商品推荐、多语言沟通、知识库、质检培训和跨部门交接。适用于由客售前后资深经理接管复杂经营任务、制定季度或月度计划、跨专项协同、经营诊断和复盘，交付角色经营目标与责任边界、专项 Skill 编排与决策、经营节奏、风险和团队行动计划。
- [客售前后资深经理周报](客服角色/客售前后资深经理/senior-customer-service-presales-manager-weekly-report/README.md)：基于真实经营证据生成客售前后资深经理视角的结论先行周报，并跟踪目标、驱动、风险、决策和责任闭环。适用于客售前后资深经理本周经营复盘、本周与上周或目标对比、管理层同步和跨团队行动跟踪，交付角色经营周报、指标变化和驱动解释、风险、决策与下周行动表。
- [跨境客售前后运营专家](客服运营/customer-service-voice-manager/README.md)：端到端完成购买前咨询、商品推荐、订单中服务、售后处理、多语言沟通、知识沉淀、质检培训和跨部门交接。适用于邮件、聊天和平台消息的售前售中售后接待、商品推荐、订单查询、异议、退款退货与多语言沟通、客服知识库、投诉升级、线索交接和质量改进，交付客售前后答复与处理方案、客服 SOP、FAQ、话术、权限和升级矩阵、咨询、售后、质检、培训与 VOC 改进报告。
