# 仓储库存主角色

## 分类定位

沉淀供应链资深经理、仓储库存资深经理及各自专属周报 Skill。

## 选择原则

- 先确认货主、仓库、库存状态、实物时间点和系统事实源。
- 仓储库存负责补货后的实物与库存执行，渠道运营提供需求和承诺，市场采购负责供应与采购。
- 实物、单据和系统状态不一致时必须隔离、对账和留痕，不得无证据调平。

## 当前 Skill

- [仓储库存资深经理](仓储库存资深经理/senior-warehouse-inventory-manager/README.md)：统筹多仓库存准确性、入出库、库内作业、库存分配、退货逆向、仓容安全、成本和履约 SLA。适用于由仓储库存资深经理接管复杂经营任务、制定季度或月度计划、跨专项协同、经营诊断和复盘，交付角色经营目标与责任边界、专项 Skill 编排与决策、经营节奏、风险和团队行动计划。
- [仓储库存资深经理周报](仓储库存资深经理/senior-warehouse-inventory-manager-weekly-report/README.md)：基于真实经营证据生成仓储库存资深经理视角的结论先行周报，并跟踪目标、驱动、风险、决策和责任闭环。适用于仓储库存资深经理本周经营复盘、本周与上周或目标对比、管理层同步和跨团队行动跟踪，交付角色经营周报、指标变化和驱动解释、风险、决策与下周行动表。
- [供应链资深经理](供应链资深经理/senior-supply-chain-manager/README.md)：统筹供应商、质量、采购、库存、仓储、运输、履约和供应链现金风险。适用于由供应链资深经理接管复杂经营任务、制定季度或月度计划、跨专项协同、经营诊断和复盘，交付角色经营目标与责任边界、专项 Skill 编排与决策、经营节奏、风险和团队行动计划。
- [供应链资深经理周报](供应链资深经理/senior-supply-chain-manager-weekly-report/README.md)：基于真实经营证据生成供应链资深经理视角的结论先行周报，并跟踪目标、驱动、风险、决策和责任闭环。适用于供应链资深经理本周经营复盘、本周与上周或目标对比、管理层同步和跨团队行动跟踪，交付角色经营周报、指标变化和驱动解释、风险、决策与下周行动表。
