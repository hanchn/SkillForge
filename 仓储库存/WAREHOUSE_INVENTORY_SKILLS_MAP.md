# 仓储库存 Skill 地图

## 一级定位

仓储库存负责商品进入采购交付之后的库存计划、实物入库、库内控制、渠道分配、出库履约、运输协同、退货逆向和仓储安全。所有结论必须同时对齐实物、业务单据和 IMS/WMS/OFS/OMS/TMS 状态。

## 主角色

- `供应链资深经理`：统筹供应、采购交接、库存、仓储、运输、履约、成本和供应韧性。
- `仓储库存资深经理`：统筹库存准确、入出库、仓内作业、分配、逆向、仓容安全和仓储 SLA。
- 两个主角色均有专属周报 Skill。

## 专项能力

| 环节 | Skill | 核心职责 |
|---|---|---|
| 补货计划与预警 | `inventory-replenishment-planner` | 监测、告警、需求、提前期、安全库存、补货测算、采购交接和关闭复盘 |
| 入库 | `warehouse-inbound-operations-manager` | 预约、收货、差异、质检交接、上架和状态生效 |
| 出库 | `warehouse-outbound-fulfillment-manager` | 波次、拣选、复核、包装、称重、面单和交接 |
| 库存控制 | `inventory-control-cycle-count-manager` | 盘点、冻结、差异、调整、损耗和审计 |
| 库存分配 | `inventory-allocation-planner` | 多仓多渠道配额、调拨、服务水平和超卖控制 |
| 履约物流 | `ecommerce-fulfillment-logistics-planner` | 仓配、承运、时效、费用、轨迹和异常 |
| 逆向物流 | `returns-reverse-logistics-manager` | 退货、检验、分级、再售、报废和追偿 |
| 仓容安全 | `warehouse-safety-capacity-manager` | 容量、动线、设备、消防、危险品、峰值和连续性 |

## 跨部门边界

- 市场采购负责供应商、采购订单、样品、成本、交期和采购质量；仓储库存从到货预约与库存计划开始接管实物执行。
- 补货预警由仓储库存产生风险队列和建议窗口，但不自动形成采购承诺；采购订单仍由市场采购按预算、MOQ、供应和审批执行。
- 渠道运营提供渠道需求、促销计划和客户承诺；仓储库存返回可售、缺货风险、分配结果和履约能力。
- 产品架构 Skill 定义 IMS/WMS/OFS/TMS 的系统边界，本目录 Skill 负责业务作业和经营控制。
