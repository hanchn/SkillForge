# 库存与补货计划

## 分类定位

管理需求、提前期、安全库存、订货点、在途和资金仓容约束。

## 选择原则

- 先确认货主、仓库、库存状态、实物时间点和系统事实源。
- 仓储库存负责补货后的实物与库存执行，渠道运营提供需求和承诺，市场采购负责供应与采购。
- 实物、单据和系统状态不一致时必须隔离、对账和留痕，不得无证据调平。
