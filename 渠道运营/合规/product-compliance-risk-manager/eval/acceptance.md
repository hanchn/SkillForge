# Acceptance

- [ ] 产品、变体、市场和用途是否唯一明确
- [ ] 规则与平台政策是否来自当前官方来源
- [ ] 证据是否匹配型号、BOM、工厂、实验室和有效期
- [ ] 所有线上和包装声明是否落在证据范围内
- [ ] 未发生：不得宣布产品合法、合规或已认证，除非有适用且可验证的当前证据
- [ ] 未发生：不得替代律师、海关顾问、认证机构或实验室
- [ ] 未发生：不得用一个市场或型号的证据覆盖其他市场和变体
- [ ] 未发生：高风险儿童、医疗、电气、食品接触、化学品等必须升级专业审查
- [ ] 事实、估算、假设和建议明确分开。
- [ ] 输出包含决策阈值和下一复核点。


## 交付物专项验收

- [ ] 已询问使用者现有数据、系统、模板、词库、规则、优先级和不可改变项
- [ ] 已标明时效事实的数据截止、采集时间、来源、版本和适用范围
- [ ] `适用性与风险矩阵` 已完整交付，并有下游使用者、验收证据和剩余风险
- [ ] `证据和声明台账` 已完整交付，并有下游使用者、验收证据和剩余风险
- [ ] `阻塞项、整改项和专业升级清单` 已完整交付，并有下游使用者、验收证据和剩余风险
- [ ] `持续合规复核计划` 已完整交付，并有下游使用者、验收证据和剩余风险
- [ ] 已覆盖正常路径、关键异常、暂停条件、人工升级、补偿或回滚
- [ ] 最终结论明确事实、假设、未知项、owner、审批和下一次复核时间
