# Acceptance

- [ ] 收入是否扣除折扣、退款和税口径差异
- [ ] 所有变动成本是否按相同单位和币种归一
- [ ] break-even 指标能否由模型复算
- [ ] 现金高峰是否考虑采购、运输、入仓和回款时点
- [ ] 未发生：不得遗漏退货、广告、履约、仓储和折扣后仍称净利润
- [ ] 未发生：不得把 GMV、销售额、毛利和现金流混为同一口径
- [ ] 未发生：平台费用和税率会变化，必须注明来源日期和适用市场
- [ ] 未发生：没有可靠数据时输出范围与补证计划，不提供伪精确利润
- [ ] 事实、估算、假设和建议明确分开。
- [ ] 输出包含决策阈值和下一复核点。


## 交付物专项验收

- [ ] 已询问使用者现有数据、系统、模板、词库、规则、优先级和不可改变项
- [ ] 已标明时效事实的数据截止、采集时间、来源、版本和适用范围
- [ ] `单位经济模型` 已完整交付，并有下游使用者、验收证据和剩余风险
- [ ] `情景与敏感性表` 已完整交付，并有下游使用者、验收证据和剩余风险
- [ ] `资金占用与盈亏平衡结论` 已完整交付，并有下游使用者、验收证据和剩余风险
- [ ] `利润改进优先级` 已完整交付，并有下游使用者、验收证据和剩余风险
- [ ] 已覆盖正常路径、关键异常、暂停条件、人工升级、补偿或回滚
- [ ] 最终结论明确事实、假设、未知项、owner、审批和下一次复核时间
