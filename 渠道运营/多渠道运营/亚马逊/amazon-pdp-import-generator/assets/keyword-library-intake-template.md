## 2. 是否优先参考给定关键词

由使用者选择，不得因上传关键词库而默认优先：

- `REQUIRED_CANDIDATE`：每个给定词都必须评估并说明采用或排除；不代表必须写入。
- `PREFERRED`：相关性、事实支持和字段适配相同时，优先布局给定词。
- `REFERENCE_ONLY`：只用于扩词、同义词、趋势对照和研究，不提高布局优先级。
- `NO_PRIORITY`：与本次市场、趋势和商品事实证据统一评分。

## 3. 优先级护栏

- 用户优先词必须与当前商品、站点、类目和目标客户直接相关。
- 不得写入竞品商标、禁止词、误导性词、无法证明的属性或声明。
- 不得为满足优先要求破坏自然表达、字段限制、去重和移动端可读性。
- 被排除的用户词必须记录原因，不得静默丢弃。
