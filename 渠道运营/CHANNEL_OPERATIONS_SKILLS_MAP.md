# 渠道运营 Skill 地图

## 一级定位

渠道运营负责“在具体渠道把可售商品稳定、合规、盈利地经营起来”，覆盖渠道策略、平台店铺、商品发布、定价、活动、转化、客户体验和经营健康。

## 主角色

- 跨境电商资深经理：统筹跨渠道战略、利润、现金和组织节奏。
- 亚马逊运营资深经理：对 Amazon 店铺经营结果负责。
- 独立站运营资深经理：对 DTC 站点经营结果负责。
- TikTok Shop 运营资深经理：对内容电商店铺经营结果负责。
- 客户体验资深经理、业务数据分析资深经理。

每个角色目录同时包含主角色 Skill 和专属周报 Skill。

## 专项能力链

| 环节 | 专项能力 |
|---|---|
| 商品经营 | 平台 Listing、商品发布、定价与单位经济 |
| 渠道经营 | 渠道组合、Amazon、Shopify、TikTok Shop、产品上下架、CRO |
| 客户与健康 | 评价声誉、退款售后、渠道体验和 VOC；购买前咨询由客服售前承接 |
| 经营控制 | 收入、利润、渠道效率和经营复盘 |

## 与精准营销的接口

- 渠道运营提供可售商品、价格、库存、落地页、利润护栏和履约承诺。
- 精准营销提供目标人群、流量、内容、创意、活动和触达计划。
- 双方共同对转化和客户价值负责，但预算归因、库存风险和利润口径必须有明确 owner。

## 与市场采购、仓储库存的接口

- 渠道运营提交市场需求、目标价格、销量节奏、活动计划和客户承诺。
- 市场采购负责机会验证、选品、样品、供应商、采购成本与采购交付。
- 仓储库存负责补货、入出库、库存准确、渠道分配、仓配与逆向。
- 渠道运营消费可售库存和履约承诺，不直接修改仓储库存事实或替代采购审批。

## 与客服售前的接口

- 渠道运营提供商品、价格、活动、库存展示、交易政策和渠道规则的有效口径。
- 客服售前负责购买前咨询、商品推荐、多语言沟通、知识库、线索分级和质检培训。
- 评价声誉、退款售后与渠道体验仍由渠道运营负责；跨阶段问题必须保留会话、订单和责任交接记录。
