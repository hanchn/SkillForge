# 渠道运营 Skill 地图

## 一级定位

渠道运营负责“把商品稳定、合规、盈利地卖出去并完成交付”，覆盖渠道策略、平台店铺、商品、选品、供应链、库存、物流、定价、履约、客户体验和经营健康。

## 主角色

- 跨境电商业务负责人：统筹跨渠道战略、利润、现金和组织节奏。
- 亚马逊运营高级经理：对 Amazon 店铺经营结果负责。
- 独立站运营高级经理：对 DTC 站点经营结果负责。
- TikTok Shop 运营高级经理：对内容电商店铺经营结果负责。
- 选品与商品高级经理、供应链高级经理、客户体验高级经理、业务数据分析高级经理。

每个角色目录同时包含主角色 Skill 和专属周报 Skill。

## 专项能力链

| 环节 | 专项能力 |
|---|---|
| 机会与商品 | 选品、商品组合、平台 Listing、定价与单位经济 |
| 渠道经营 | 渠道组合、Amazon、Shopify、TikTok Shop、产品上下架、CRO |
| 供给与交付 | 供应商、质量、合规、库存、补货、物流和履约 |
| 客户与健康 | 多语言客服、评价声誉、退款售后和 VOC |
| 经营控制 | 收入、利润、库存、渠道效率和经营复盘 |

## 与精准营销的接口

- 渠道运营提供可售商品、价格、库存、落地页、利润护栏和履约承诺。
- 精准营销提供目标人群、流量、内容、创意、活动和触达计划。
- 双方共同对转化和客户价值负责，但预算归因、库存风险和利润口径必须有明确 owner。
