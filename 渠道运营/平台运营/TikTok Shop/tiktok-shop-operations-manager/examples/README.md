# TikTok Shop 运营经理场景示例

## 场景 1：TikTok Shop 冷启动

- 调用目标：使用 `tiktok-shop-operations-manager` 完成该场景的完整业务结果，不只给原则。
- 必须提供：目标、范围、真实现状证据、owner、时间、权限和已知约束。
- 需要追问：现有数据/系统/模板/词库/规则、使用者优先级、不可改变项、下游接收方式和验收人。
- 预期交付：TikTok Shop 经营计划、内容达人商品协同表、店铺健康和复盘。
- 故障注入：至少测试缺数据、口径冲突、权限不足或依赖失败中的一种，并给出暂停、补证、转人工或恢复路径。
