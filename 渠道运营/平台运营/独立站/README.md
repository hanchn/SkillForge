# 独立站运营

## 分类定位

覆盖 Shopify 商品、流量、转化、订单和客户经营。

## 选择原则

- 主角色 Skill 负责目标、取舍、协同和验收，专项 Skill 负责深度执行。
- 所有增长动作同时受利润、库存、合规、履约和客户体验约束。
- 数字结论必须组合数据看板或财务口径验证。

## 当前 Skill

- [Shopify 独立站运营经理](shopify-store-operations-manager/README.md)：统筹商品、内容、流量、转化、订单、客户和技术健康的独立站经营。适用于Shopify 日常运营、新品与活动上线、独立站增长和问题排查，交付独立站经营计划、上线与健康检查表、渠道和站内经营复盘。
