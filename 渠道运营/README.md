# 渠道运营

## 分类定位

以平台、店铺、商品、商业化和客户体验为核心，对渠道收入、利润和经营健康负责。参见 [渠道运营 Skill 地图](CHANNEL_OPERATIONS_SKILLS_MAP.md)。

## 选择原则

- 主角色 Skill 负责目标、取舍、协同和验收，专项 Skill 负责深度执行。
- 所有增长动作同时受利润、库存、合规、履约和客户体验约束。
- 数字结论必须组合数据看板或财务口径验证。
