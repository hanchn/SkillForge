# 渠道运营主角色

## 分类定位

沉淀渠道经营主角色 Skill，并由主角色编排各专项执行 Skill。

## 选择原则

- 主角色 Skill 负责目标、取舍、协同和验收，专项 Skill 负责深度执行。
- 所有增长动作同时受利润、库存、合规、履约和客户体验约束。
- 数字结论必须组合数据看板或财务口径验证。
