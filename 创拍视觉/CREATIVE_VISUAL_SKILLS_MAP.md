# 创拍视觉 Skill 地图

## 一级定位

创拍视觉是覆盖创意、拍摄、视觉与广告素材生产的独立职能，负责把商品事实、品牌策略和渠道需求转化为可策划、可拍摄、可生成、可验收、可授权、可投放和可复用的图片与视频资产。精准营销提出人群、信息、渠道和效果需求；创拍视觉负责创意方案、制作产能、视觉质量、广告素材真实性与交付。

## 主角色

- `创拍视觉资深经理`：统筹需求组合、创意策略、传统拍摄、AI 制作、人员供应商、预算、质量和素材资产。
- `创拍视觉资深经理周报`：报告需求、产能、预算、交付、质量、授权、素材表现和下周决策。

## 传统创意制作

| 环节 | Skill | 核心职责 |
|---|---|---|
| 创意策略 | `creative-strategy-producer` | 人群洞察、产品证据、角度、脚本和测试编码 |
| 制片统筹 | `shooting-production-coordinator` | 预算、排期、样品、人员、场地、设备和现场安全 |
| 商品摄影 | `product-photography-director` | 主图、细节、场景、模特、色彩与真实性 |
| 综合制作 | `product-photo-video-production-manager` | 图片视频项目总体方案、资源、验收和交付 |
| 短视频 | `short-form-video-producer` | 钩子、脚本、分镜、拍摄、声音和版本测试 |
| UGC/达人制作 | `ugc-content-production-manager` | 创作者、寄样、brief、反馈、授权和交付 |
| 后期交付 | `post-production-delivery-manager` | 选片、剪辑、修图、调色、字幕、本地化和导出 |
| 素材运营 | `creative-asset-operations-manager` | 标签、版本、权利、分发、效果回流和复用 |

## AI 创意制作

| 能力 | Skill | 核心职责 |
|---|---|---|
| 虚拟模特 | `virtual-model-content-producer` | 模特生成/替换、虚拟试穿、人物授权和商品一致性 |
| AIGC 制作 | `ai-image-video-production-manager` | 文生图、图生图、文生视频、图生视频、局部修改、批量变体和人工验收 |
| 智能打标 | `creative-asset-intelligent-tagging-specialist` | AI 打标工具规划、标签体系、模型推理、人审、CMS/DAM 回写、金标评估、监控、版本和效果标签 |
| 智能剪辑 | `intelligent-video-editing-specialist` | 镜头识别、粗剪、字幕、画幅、多语言和批量交付 |
| 真实性合规 | `ai-creative-authenticity-compliance-reviewer` | 商品失真、人物权利、版权、披露、平台和生成血缘 |

## Tool 与 Skill 的边界

- 模型、API、摄影设备、剪辑软件和资产管理系统属于 Tool。
- Skill 定义何时使用哪个 Tool、输入输出、真实性标准、人工复核、授权、验收和回滚。
- AI 结果不能自动当作真实拍摄或商品证据；关键商品属性和人物权利必须人工确认。
