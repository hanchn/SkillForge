- 调用目标：使用 `post-production-delivery-manager` 完成该场景的完整业务结果，不只给原则。
- 必须提供：目标、范围、真实现状证据、owner、时间、权限和已知约束。
- 需要追问：现有数据/系统/模板/词库/规则、使用者优先级、不可改变项、下游接收方式和验收人。
- 预期交付：后期任务与版本矩阵、修图剪辑、字幕和声音规范、质检、命名、归档和交付清单。
