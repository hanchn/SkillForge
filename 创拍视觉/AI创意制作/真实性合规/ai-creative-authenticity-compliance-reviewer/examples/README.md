# AI 创意真实性与合规审查师场景示例

## 场景 1：AI 商品图视频上线前审查

- 调用目标：使用 `ai-creative-authenticity-compliance-reviewer` 完成该场景的完整业务结果，不只给原则。
- 必须提供：目标、范围、真实现状证据、owner、时间、权限和已知约束。
- 需要追问：现有数据/系统/模板/词库/规则、使用者优先级、不可改变项、下游接收方式和验收人。
- 预期交付：AI 创意逐项审查表、失真、权利和平台风险分级、修改、披露、留档和禁止上线清单。
- 故障注入：至少测试缺数据、口径冲突、权限不足或依赖失败中的一种，并给出暂停、补证、转人工或恢复路径。
