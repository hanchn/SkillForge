# 虚拟模特

## 分类定位

管理人像生成、模特替换、虚拟试穿、授权和商品一致性。

## 选择原则

- 主角色负责需求、产能、预算、质量和交付，专项 Skill 负责策略、拍摄、AI 制作、后期或资产治理。
- 所有素材必须锁定商品事实、使用范围、授权、版本和质量门禁。
- AI 生成、虚拟模特和智能剪辑必须保留模型与修改血缘，并经过人工真实性和合规复核。

## 当前 Skill

- [虚拟模特内容制作专家](virtual-model-content-producer/README.md)：使用合规的人像生成、模特替换或虚拟试穿工具制作跨人群商品内容，并严格保持商品结构与真实属性。适用于服饰、配件和生活方式虚拟模特图、多地区模特与场景版本、真人素材的合规模特替换，交付虚拟模特制作 brief、身份、姿态、商品一致性和版本方案、真实性、授权、偏差和交付验收报告。
