# 智能打标验收

- [ ] 输入素材在授权范围内，外部模型、数据传输和保存边界已确认
- [ ] 正式生成前已询问是否存在关键词/标签库，并记录版本、语言、市场、来源、owner 和更新时间
- [ ] 已询问是否优先参考给定关键词，并记录 `REQUIRED_CANDIDATE`、`PREFERRED`、`REFERENCE_ONLY` 或 `NO_PRIORITY`
- [ ] 优先关键词只有在素材有直接证据且允许标签库接受时输出，没有提高置信度或绕过受限/禁止规则
- [ ] 所有自动标签均来自允许标签库，且有直接视觉或已授权元数据证据
- [ ] 受限标签全部进入人工审核，禁止标签未被输出
- [ ] 人物素材未做人脸身份、敏感属性、精确年龄或社会属性推断
- [ ] 疑似未成年人、正脸、Logo、水印和第三方文字已强制转人工
- [ ] 每个标签包含置信度、证据和来源，阈值规则执行正确
- [ ] 输出通过 JSON Schema，记录 Skill、模型、标签库、素材和审核版本
- [ ] 批量回写具备预览、幂等、审计、权限和回滚
- [ ] 明确标注 Skill 无法替代的权限、授权真实性和不可篡改审计控制


## 交付物专项验收

- [ ] 已询问使用者现有数据、系统、模板、词库、规则、优先级和不可改变项
- [ ] 已标明时效事实的数据截止、采集时间、来源、版本和适用范围
- [ ] `关键词/标签库接入与优先级确认记录` 已完整交付，并有下游使用者、验收证据和剩余风险
- [ ] `AI 打标能力、用户流程与标签体系` 已完整交付，并有下游使用者、验收证据和剩余风险
- [ ] `模型、置信度、人审决策和批量回写方案` 已完整交付，并有下游使用者、验收证据和剩余风险
- [ ] `金标评估、监控、错误集、版本、回滚和演进报告` 已完整交付，并有下游使用者、验收证据和剩余风险
- [ ] `决策、执行、验收、风险和未知项记录` 已完整交付，并有下游使用者、验收证据和剩余风险
- [ ] 已覆盖正常路径、关键异常、暂停条件、人工升级、补偿或回滚
- [ ] 最终结论明确事实、假设、未知项、owner、审批和下一次复核时间
