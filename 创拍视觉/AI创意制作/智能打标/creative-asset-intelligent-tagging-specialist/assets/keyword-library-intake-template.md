# 关键词与标签库生成前确认

> 本确认必须在正式生成标签前完成。未回答时只能做素材检查、字段盘点和调研，不能输出正式标签结果。

## 2. 是否优先参考使用者提供的关键词

选择一项并由使用者确认：

- `REQUIRED_CANDIDATE`：优先检查每个给定词；只有素材有直接证据且合规允许时才能输出，不能强制命中。
- `PREFERRED`：同等证据和置信度下优先选择给定词。
- `REFERENCE_ONLY`：仅用于召回、同义词和搜索，不提高标签置信度。
- `NO_PRIORITY`：完全按批准标签库、素材证据和模型阈值决定。

## 3. 冲突处理

- 用户词不在允许标签库：不自动新增，提交 taxonomy owner 审批。
- 用户词命中受限标签：`REVIEW_REQUIRED`。
- 用户词命中禁止标签或要求敏感推断：`COMPLIANCE_REJECTED`。
- 用户词与素材无直接证据：不输出，并记录 `NO_VISUAL_EVIDENCE`。
- 多个词同义或冲突：按已批准 canonical tag 映射；无法裁决则转人工。
