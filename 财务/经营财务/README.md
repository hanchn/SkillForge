# 经营财务

## 分类定位

连接财务账、商品渠道单位经济和经营决策。

## 选择原则

- 先确认主体、期间、币种、账簿和法定/管理口径。
- 跨境财务高级经理负责总体结果，专项 Skill 负责关账、预算、资金、税务或经营财务。
- 预估、未关账和管理数据不得冒充正式财务结果。

## 当前 Skill

- [经营财务与利润经理](management-accounting-profitability-manager/README.md)：连接财务账、商品渠道经营和单位经济，支持定价、预算、资源和退出决策。适用于渠道和 SKU 盈利核算、经营复盘、定价、促销和资源决策，交付管理利润口径、商品渠道盈利矩阵、差异、情景和经营建议。
