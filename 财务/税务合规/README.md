# 税务合规

## 分类定位

管理跨境 VAT、销售税、关税和税务证据。

## 选择原则

- 先确认主体、期间、币种、账簿和法定/管理口径。
- 跨境财务高级经理负责总体结果，专项 Skill 负责关账、预算、资金、税务或经营财务。
- 预估、未关账和管理数据不得冒充正式财务结果。

## 当前 Skill

- [跨境税务与 VAT 规划师](cross-border-tax-vat-planner/README.md)：按主体、交易流、货物流和销售国识别 VAT、销售税、关税、所得税和申报证据要求。适用于进入新市场税务评估、VAT/IOSS/销售税流程、税务申报和审计证据准备，交付交易与税务责任图、注册申报和证据日历、风险、整改和专业顾问升级清单。
