# 跨境运营营销 Skill 地图

## 团队能力链

| 阶段 | 核心能力 | 主要 Skill |
|---|---|---|
| 战略与机会 | 市场情报、选品、品牌定位、上市 | `cross-border-market-intelligence`、`ecommerce-product-research`、`brand-positioning-strategist`、`go-to-market-launch-planner` |
| 商品与供给 | 商品信息、供应商、合规、定价、库存、物流 | `marketplace-listing-optimizer`、`supplier-sourcing-quality-manager`、`product-compliance-risk-manager`、`pricing-promotion-optimizer`、`inventory-replenishment-planner` |
| 渠道经营 | 渠道组合、Amazon、Shopify、TikTok Shop | `channel-portfolio-strategist`、三个店铺运营经理 Skill |
| 获客与内容 | SEO、GEO、内容、社媒、达人、广告、创意 | `seo-optimizer`、`geo-optimizer`、`content-marketing-planner`、`social-media-operations-manager`、`influencer-affiliate-manager`、`commerce-advertising-optimizer`、`creative-strategy-producer` |
| 转化与活动 | 店铺转化、商品发布、活动、邮件 | `shopify-conversion-auditor`、`storefront-product-publishing-manager`、`promotion-calendar-manager`、`email-lifecycle-marketer` |
| 客户与增长 | 客服、评价、留存、归因、全链路增长 | `customer-service-voice-manager`、`review-reputation-manager`、`customer-retention-loyalty-manager`、`marketing-attribution-planner`、`cross-border-growth-operator` |
| 经营控制 | 单位经济、经营和业务数据分析 | `ecommerce-unit-economics-analyst` 与 `数据/业务分析/` Skill 家族 |

## 使用原则

- 店铺运营经理是端到端 owner；专项 Skill 是专业执行者，两者通过目标、输入物和验收对接。
- 增长负责人不能绕过利润、库存、合规、履约和客户体验护栏。
- 渠道后台数据不是公司唯一事实源，预算和利润决策需组合业务分析 Skill。
- 所有国家、平台规则和广告能力在执行前应以当前官方资料和真实账户为准。
