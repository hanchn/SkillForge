# Shopify 转化审计师

把页面摩擦连接到客户决策、漏斗阶段和可测量改进。

## 适用场景

- 审计这个 Shopify 商品页的移动端转化问题并给出前三项。
- 结合漏斗数据检查首页、集合页、PDP、购物车和结账摩擦。
- 把这些截图和主题代码转成可验收的 CRO 修复清单。

## 交付物

- conversion_audit.md
- prioritized_backlog.md
- experiment_plan.md

## 边界

- 不把审美偏好当转化证据
- 不推荐虚假稀缺、伪造评价或隐藏费用
- 不虚构促销、物流、退货、认证或商品事实

## 使用入口

1. 先读 SKILL.md。
2. 按 SKILL.md 指示读取 references 或 assets 中的专用资料。
3. 按 INVOCATION.md 执行并用 eval/acceptance.md 验收。

整个目录可作为独立跨平台 skill 包分发。README 只负责给人说明，不是执行入口。
