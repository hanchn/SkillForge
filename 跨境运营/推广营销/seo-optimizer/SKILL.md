# SEO营销优化技能

## Identity

- skill id: `seo-optimizer`
- display name: `SEO营销优化技能`
- type: `portable-business-skill`
- scope: `cross-platform`

## What It Does

- 分析目标文本或产品，挖掘高流量、低竞争的 SEO 关键词
- 重写产品标题（Title）、描述（Description）以符合 SEO 规范
- 提供 H1/H2 标签布局与页面内容结构优化建议

## When To Use

- 需要将常规内容改写为利于搜索引擎收录的格式
- 需要为新产品或文章寻找核心关键词
- 需要优化现有页面的转化率与自然搜索排名
