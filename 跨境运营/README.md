## 组合原则

- 店铺运营 Skill 负责端到端结果，专项 Skill 负责深度专业任务。
- 增长动作必须同时受利润、合规、库存、履约和客户体验约束。
- 需要数字结论时组合 `数据/业务分析/` Skill，不让运营判断替代数据验证。
