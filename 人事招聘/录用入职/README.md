# 录用入职

## 分类定位

管理录用审批、Offer、背景核验和入职交接。

## 选择原则

- 先确认业务目标、组织范围、工作地、用工主体、编制和岗位成功标准。
- 人事招聘资深经理负责目标与编排，专项 Skill 负责规划、寻访、面试、录用和员工流程。
- 坚持岗位相关、数据最小化和一致标准；劳动法律事项升级法务和当地专业人士。

## 当前 Skill

- [录用与入职管理专员](offer-onboarding-manager/README.md)：管理候选人审批、Offer、背景核验、合同资料、入职准备和首日到试用期交接。适用于候选人录用审批、Offer 谈判和签署、跨部门入职和试用期启动，交付录用审批与 Offer 清单、入职准备责任表、首日、30/60/90 天交接计划。
