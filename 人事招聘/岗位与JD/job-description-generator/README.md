# 招聘需求与 JD 自动生成

> skill id：job-description-generator  
> 当前版本：1.1.0  
> 产品状态：可用  
> 所属分类：人事招聘 / 岗位与JD

## 产品定位

在发起人创建招聘需求时，一次生成候选人版 JD、HR 精准岗位培训文档、内部岗位说明书和机器可读筛选规则。发起人只需修改确认，HR 接收 `READY_FOR_HR` 招聘包后无需再次沟通岗位。

## 自动化原则

- 正常路径自动生成、自动同步、零沟通交接。
- 非关键未知项保留为待确认，不阻塞可编辑草案。
- 关键冲突、未经批准承诺和不当筛选条件才进入 `REVIEW_REQUIRED`。
- 文案修改只更新 JD；岗位要求修改同步所有产物并递增 rubric 版本。
- 市场与薪酬调研按需执行，不作为所有招聘需求的强制前置。

## 适用场景

- 创建招聘需求并自动生成完整招聘包
- 发起人修改 JD 后同步筛选规则
- 将现有 JD 迁移成 HR 培训与机器可读筛选契约
- 按需执行岗位市场与薪酬调研

## 输入与输出

最低输入为岗位名称、招聘原因、主要结果和发起人；建议同时提供部门、人数、职级、地点、用工方式和到岗时间。输出可编辑 JD、HR 培训、内部岗位说明、`screening-spec.json`、状态、版本和异常清单。

## 下游

`local-resume-screening-compliance-specialist` 直接消费已确认筛选契约，无需重新解读自然语言 JD。
