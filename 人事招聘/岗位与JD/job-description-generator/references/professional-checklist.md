# 招聘需求自动生成专业检查表

- [ ] 招聘需求已生成唯一 `request_id`、`package_version` 和 `rubric_version`
- [ ] 正常路径没有要求发起人与 HR 进行重复口头交接
- [ ] 候选人 JD、HR 培训文档、内部岗位说明和筛选规则内容一致
- [ ] 每个标准有稳定 ID、类别、岗位相关理由、关键词/同义词、证据、替代证据和核验方式
- [ ] 关键词只用于召回，没有用出现次数直接证明能力
- [ ] 可培养项没有被升级成自动硬门槛
- [ ] 敏感属性和无关代理变量已进入禁用清单
- [ ] 发起人修改已分类为 `COPY_ONLY / REQUIREMENT_CHANGE / COMPLIANCE_CHANGE`
- [ ] 实质要求变化已递增 rubric 版本并使旧筛选结果失效
- [ ] 无阻断项输出 `READY_FOR_HR`，有阻断项只给最少修正字段
- [ ] 市场与薪酬调研仅在明确需要时执行，并有当前来源、样本和口径
