# 人事招聘 Skill 地图

## 定位

覆盖从业务目标、组织编制、岗位定义到寻访、面试、录用、入职、培训发展、异动、离职和员工关系的完整人才生命周期。主角色负责目标、取舍和协同，专项 Skill 提供可复用的深度执行流程。

## 角色层

- `人事招聘资深经理`：统筹编制、人才供给、招聘质量、员工流程、组织风险和跨团队节奏。
- `人事招聘资深经理周报`：统一编制、漏斗、Offer、入职、离职、员工事项和待决策口径。

## 公共入口

- `人事招聘通用执行专家`：当任务尚未明确归属时，先完成范围、权限、证据和风险识别，再路由到专项 Skill。

## 专项层

- 人力编制与招聘规划：业务目标到能力缺口、岗位优先级和预算。
- 岗位说明书与招聘 JD：岗位使命、职责边界、能力证据、发布稿和合规检查。
- 本地简历筛选与合规评估：读取授权本地简历，按 JD 岗位证据输出匹配、缺口、核验问题、异常原因和人工复核分层，不自动拒绝或录用。
- 候选人寻访运营：人才画像、渠道组合、人才地图、触达和人才库。
- 结构化面试与评估：岗位证据、工作样本、评分锚点、校准和偏见控制。
- 录用与入职管理：录用审批、Offer、背景核验、入职准备和试用期交接。
- 员工生命周期与档案：主档、异动、合同、离职、权限回收、保留和审计。
- 绩效与员工关系：目标反馈、绩效改进、员工诉求、调查协同和沟通。
- 企业员工公告撰写：普通通知、政策公告、组织沟通草案、多渠道版本和发布检查。
- 员工培训组织与运营：培训需求、计划预算、课程讲师、通知报名、现场执行、效果评估和学习档案。

## 与法律 Skill 的边界

人事 Skill 负责业务流程、记录、体验和执行；`法律政务/劳动用工`负责适用司法辖区、合同、劳动义务、纪律、解雇、歧视和争议风险。高风险事项必须组合使用，并由当地合格专业人士复核。
