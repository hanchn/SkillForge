# 无人值守简历筛选与面试预约

> skill id：local-resume-screening-compliance-specialist  
> 当前版本：1.1.0  
> 产品状态：可用  
> 所属分类：人事招聘 / 简历筛选

## 产品定位

直接消费招聘需求阶段生成的 `screening-spec.json`，无人值守完成简历解析、岗位证据匹配、候选池、漏斗诊断、面试预约、会议室、面试官通知、候选人二次确认邮件和飞书异常告警。

## 正常路径

有效契约和授权连接器就绪后不再重复询问发起人或 HR。系统自动处理简历并预约面试，无异常输出 `AUTOMATION_COMPLETED`。

## 异常路径

低候选池、解析质量、渠道错配、单项过度拦截、规则漂移、预约 API、会议室、面试官冲突、邮件投递和候选人确认超时均生成标准飞书事件。

## 数据与决策边界

- 简历清单默认不落全文，原文件只读，敏感信息不参与匹配。
- 自动化形成候选池和面试预约，不把结果作为永久拒绝或最终录用的唯一依据。
- 未确认真实 API、权限、测试环境和回滚时只做 dry-run。
