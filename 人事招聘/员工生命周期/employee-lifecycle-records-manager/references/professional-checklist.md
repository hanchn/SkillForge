# 员工生命周期与档案管理员专业检查表

## 执行前

- [ ] 目标、范围、国家/渠道/系统、周期和责任人明确
- [ ] 已区分事实、假设、缺失数据和待确认口径
- [ ] 已读取一手资料，并记录来源时间与适用版本

## 核心维度

- [ ] 员工唯一标识和数据 owner 已定义、验证或标记为未知
- [ ] 合同、职位、汇报和成本中心 已定义、验证或标记为未知
- [ ] 假勤、证明和变更证据 已定义、验证或标记为未知
- [ ] 离职、资产和账号回收 已定义、验证或标记为未知
- [ ] 最小权限、保留、删除和审计日志 已定义、验证或标记为未知


## 失败模式

- 只有最佳实践，没有结合当前证据和约束。
- 只覆盖顺利路径，没有异常、补偿、回滚和人工介入。
- 指标同名但 scope、状态、时区、币种或归因窗口不同。
- 动作没有 owner、投入、依赖、验收或停止条件。

## 交付前

- [ ] 关键结论能追溯到证据或计算
- [ ] 备选方案、取舍和不做事项清楚
- [ ] 计划可分阶段执行并有验证与恢复路径
