# Acceptance

- [ ] locale、市场、货币和时区是否分开建模
- [ ] 所有 key、变量、复数和富文本标签是否匹配
- [ ] 长文本、伪本地化和 RTL 是否验证
- [ ] URL、metadata、canonical 和 hreflang 是否一致
- [ ] 目标语言语义是否安排合格复核人
- [ ] 未发生：不得把机器翻译通顺度当成完整本地化验收
- [ ] 未发生：不得通过字符串拼接构造需要语法变化的句子
- [ ] 未发生：不得把国家、语言、货币和时区视为同一个概念
- [ ] 未发生：不熟悉目标语言时必须标记语言质量需母语或专业复核
- [ ] 每项问题都有可复现证据、影响和复测方式。
