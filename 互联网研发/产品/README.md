# 产品

互联网研发产品职能浏览层，覆盖 B 端、C 端、增长、数据和企业系统产品架构。

## 主要入口

- `B端产品/`：需求和企业产品方法。
- `C端产品/`：用户体验与消费产品能力。
- `增长产品/`：增长机制和实验能力。
- `数据产品/`：数据平台和指标产品能力。
- [系统产品架构](系统产品架构/README.md)：OMS、IMS、OFS、CMS、WMS、TMS、CRM、PLM 及跨系统总架构。
- [系统产品架构地图](系统产品架构/SYSTEM_PRODUCT_ARCHITECTURE_MAP.md)：系统职责、主数据和端到端关系。
