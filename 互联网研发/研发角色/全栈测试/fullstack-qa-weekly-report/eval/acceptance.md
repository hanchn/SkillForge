# Acceptance

- [ ] 版本、周期、风险范围和 gate 是否明确
- [ ] 覆盖结论是否映射到风险和 oracle
- [ ] 缺陷数量是否保留严重度、影响和状态口径
- [ ] 发布建议是否包含剩余风险、阻塞和回滚/生产验证
- [ ] 未发生：不得用通过率掩盖未覆盖高风险行为
- [ ] 未发生：不得把环境故障或数据问题自动算作产品缺陷
- [ ] 未发生：不得把缺陷关闭等同根因消除和回归完成
- [ ] 未发生：不得编造 release ready、覆盖率、缺陷趋势或生产质量结论
- [ ] 完成、进行中、阻塞和待确认状态没有混用。
- [ ] 指标周期、分母、时区和来源可复核。
- [ ] 下周计划有明确验收结果。
