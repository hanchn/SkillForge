# Acceptance

- [ ] 周期、时区和项目范围是否明确
- [ ] 每项完成事项是否有合并/发布/验证证据
- [ ] 同一目标的跨层改动是否合并表达
- [ ] 风险和下周计划是否包含依赖与可验收结果
- [ ] 未发生：不得把 commit、PR 或代码行数直接等同业务产出
- [ ] 未发生：不得把已编码、已提测、已合并和已上线混为完成
- [ ] 未发生：不得编造进度百分比、发布日期、指标改善或责任归属
- [ ] 未发生：对外周报避免泄露凭据、内部敏感架构和不必要技术噪音
- [ ] 完成、进行中、阻塞和待确认状态没有混用。
- [ ] 指标周期、分母、时区和来源可复核。
- [ ] 下周计划有明确验收结果。
