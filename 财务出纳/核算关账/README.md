# 核算关账

## 分类定位

管理平台、支付、银行、库存和总账关账对账。

## 选择原则

- 先确认主体、期间、币种、账户、账簿和授权边界。
- 跨境财务高级经理负责会计治理，高级出纳负责授权范围内的资金执行。
- 申请、审批、付款、记账和对账必须职责分离；预估和未达账必须明确标记。

## 当前 Skill

- [财务关账与对账经理](financial-close-reconciliation-manager/README.md)：建立从平台、支付、订单、库存、银行到总账的可复核关账和对账流程。适用于月结关账、平台支付与银行对账、差异和审计证据治理，交付关账日历和责任矩阵、对账差异桥、调整、审批和证据包。
