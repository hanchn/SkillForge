# 银行与出纳对账

## 分类定位

管理银行账户、日记账、余额和未达账项。

## 选择原则

- 先确认主体、期间、币种、账户、账簿和授权边界。
- 跨境财务高级经理负责会计治理，高级出纳负责授权范围内的资金执行。
- 申请、审批、付款、记账和对账必须职责分离；预估和未达账必须明确标记。

## 当前 Skill

- [银行账户与出纳对账专员](bank-account-reconciliation-cashier/README.md)：维护银行账户台账、日记账和每日余额核对，及时识别未达账项、银行费用、退票和异常流水。适用于银行日记账维护、每日资金余额核对、未达账和异常流水处理，交付账户与余额台账、银行日记账核对表、未达账、差异和跟进清单。
