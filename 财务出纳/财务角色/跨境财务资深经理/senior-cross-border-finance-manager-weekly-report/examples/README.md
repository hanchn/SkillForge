# 跨境财务资深经理周报场景示例

## 场景 1：财务周度复盘

- 调用目标：使用 `senior-cross-border-finance-manager-weekly-report` 完成该场景的完整业务结果，不只给原则。
- 必须提供：目标、范围、真实现状证据、owner、时间、权限和已知约束。
- 需要追问：现有数据/系统/模板/词库/规则、使用者优先级、不可改变项、下游接收方式和验收人。
- 预期交付：财务周报、现金、利润和风险摘要、决策、责任人与下周行动。
- 故障注入：至少测试缺数据、口径冲突、权限不足或依赖失败中的一种，并给出暂停、补证、转人工或恢复路径。
