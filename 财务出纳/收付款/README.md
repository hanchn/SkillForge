# 收付款

## 分类定位

管理申请、审批、执行、回单、状态和异常闭环。

## 选择原则

- 先确认主体、期间、币种、账户、账簿和授权边界。
- 跨境财务资深经理负责会计治理，资金出纳资深经理负责授权范围内的资金执行。
- 申请、审批、付款、记账和对账必须职责分离；预估和未达账必须明确标记。

## 当前 Skill

- [收付款执行管理员](payment-receipt-operations-manager/README.md)：建立从业务申请、单据校验、授权审批、支付执行、回单归档到状态回写的收付款闭环。适用于供应商和物流付款、客户、平台和渠道收款认领、批量付款与失败重试，交付收付款 SOP 与权限矩阵、支付批次和状态台账、回单、异常和升级记录。
