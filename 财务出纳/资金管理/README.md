# 资金管理

## 分类定位

管理现金、营运资金、多币种头寸和汇率风险。

## 选择原则

- 先确认主体、期间、币种、账户、账簿和授权边界。
- 跨境财务资深经理负责会计治理，资金出纳资深经理负责授权范围内的资金执行。
- 申请、审批、付款、记账和对账必须职责分离；预估和未达账必须明确标记。

## 当前 Skill

- [现金流与营运资金经理](cash-flow-working-capital-manager/README.md)：管理现金预测、应收应付、库存资金占用、付款优先级和流动性风险。适用于十三周现金预测、营运资金改善、资金缺口和付款安排，交付现金流预测、现金转换周期分析、付款、融资和预警计划。
- [多币种资金与汇率经理](multi-currency-treasury-manager/README.md)：管理多币种账户、平台结算、汇率暴露、换汇、流动性和资金权限。适用于多币种现金管理、平台结算和换汇优化、汇率风险和资金安全治理，交付币种资金头寸、汇率暴露与情景、换汇、归集、权限和预警方案。
