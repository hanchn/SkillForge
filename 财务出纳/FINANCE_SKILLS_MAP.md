# 财务与出纳 Skill 地图

## 定位

本地图把“财务治理”和“资金执行”放在同一一级领域下，但保持职责分离：跨境财务资深经理负责核算、预算、税务、经营分析和控制；资金出纳资深经理负责授权范围内的收付款、银行账户、票据、日结与资金安全。

## 角色层

- `跨境财务资深经理`：对财务结果、口径、关账、预算、现金和风险负责。
- `跨境财务资深经理周报`：报告利润、现金、预算、关账、税务和控制事项。
- `资金出纳资深经理`：对资金执行准确性、及时性、证据和账户安全负责。
- `资金出纳资深经理周报`：报告余额、收付款、未达账、票据与支付风险。

## 专项层

- 财务：关账对账、预算预测、现金流与营运资金、跨境税务、多币种资金、经营财务。
- 出纳：收付款执行、银行与出纳对账、报销与票据、资金安全与支付权限。

## 必须遵守的控制边界

1. 申请、审批、付款、记账、对账不得由同一人全程完成。
2. 出纳不自行批准付款，不代替会计确认正式账务和税务处理。
3. 所有付款、调节和权限变更必须保留来源、审批、执行、复核和回滚证据。
4. Skill 不接收或输出真实密码、验证码、私钥和完整银行凭证。
