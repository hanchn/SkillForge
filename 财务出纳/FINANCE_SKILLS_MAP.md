# 跨境财务 Skill 地图

## 主角色

`senior-cross-border-finance-manager` 对核算、管理报表、预算、现金、税务、外汇和内控的整体质量负责；配套周报跟踪关账状态、现金风险和关键期限。

## 专项能力

| 领域 | 主要 Skill | 核心产物 |
|---|---|---|
| 核算关账 | `financial-close-reconciliation-manager` | 关账日历、对账差异桥和审计证据 |
| 预算预测 | `budget-forecast-controller` | 驱动预算、滚动预测和差异纠偏 |
| 现金营运资金 | `cash-flow-working-capital-manager` | 十三周现金预测、现金转换和预警 |
| 跨境税务 | `cross-border-tax-vat-planner` | 交易税务责任、申报日历和证据要求 |
| 多币种资金 | `multi-currency-treasury-manager` | 资金头寸、汇率暴露和权限方案 |
| 经营财务 | `management-accounting-profitability-manager` | 法定与管理口径桥、渠道 SKU 盈利矩阵 |

## 与数据分析的边界

- `数据看板/` 负责指标治理、业务分析和决策证据。
- `财务/` 负责账簿、关账、正式口径、资金、税务和控制。
- 两者通过收入、成本、退款、库存、汇率和主体口径桥对接，不能互相替代。

## 强制边界

- 不把管理口径、预估数或未关账数据冒充法定财务结果。
- 税务申报、审计意见、会计政策和资金交易必须由有权限的专业人员确认或执行。
