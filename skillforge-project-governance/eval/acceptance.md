# Acceptance

- 新 AI 能快速理解项目定位
- 新 AI 能区分项目级变更与业务级变更
- 新 AI 不再把业务 skill 放进 `.trae`
- 新 AI 知道 `README.md` 是中文产品文档和版本记录，`SKILL.md` 才是 AI 入口
- 新 AI 知道每个业务 skill 要整包可分发


## 交付物专项验收

- [ ] 已询问使用者现有数据、系统、模板、词库、规则、优先级和不可改变项
- [ ] 已标明时效事实的数据截止、采集时间、来源、版本和适用范围
- [ ] `project understanding` 已完整交付，并有下游使用者、验收证据和剩余风险
- [ ] `architecture guidance` 已完整交付，并有下游使用者、验收证据和剩余风险
- [ ] `skill package constraints` 已完整交付，并有下游使用者、验收证据和剩余风险
- [ ] 已覆盖正常路径、关键异常、暂停条件、人工升级、补偿或回滚
- [ ] 最终结论明确事实、假设、未知项、owner、审批和下一次复核时间
- [ ] 索引、版本、前端可见性、合规、深度与语义审计均已重建并保留可复核证据
