# Acceptance

## 必须通过

- [ ] 比较口径包括时间、时区、人群、粒度和过滤条件
- [ ] 数据新鲜度、重复、连接、迟到和分母漂移已检查
- [ ] 绝对变化、相对变化、分子和分母均可复核
- [ ] 驱动贡献经过加权且处理小基数
- [ ] 确认、贡献、排除和未决假设清晰分层
+- [ ] 事实、假设、推断、建议和未知项明确区分
+- [ ] 输出符合 SKILL.md 的 Output contract

## 直接失败

- [ ] 未发生：不把相关性写成因果
- [ ] 未发生：不比较不完整与完整周期而不说明
- [ ] 未发生：不以小样本百分比变化代替贡献度
+- [ ] 未用空泛最佳实践替代当前任务证据
+- [ ] 未隐藏会改变结论的重要信息缺口

## 人工抽查

- 随机选择一个关键结论，能追溯到输入证据或明确假设。
- 随机选择一个建议，能说明影响对象、执行条件和验收方式。
