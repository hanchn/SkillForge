# 数据

数据领域业务 Skill 浏览层，现已形成“指标治理与数据质量 → 专项经营分析 → 管理层复盘与预测”的完整链路。

## 入口

- [业务数据分析 Skill 地图](BUSINESS_ANALYTICS_SKILLS_MAP.md)
- `业务分析/`：经营、利润、商品、流量、漏斗、广告、客户、库存、价格、渠道和预测。
- `数据治理/`：跨系统数据质量与对账。
- `metric-diagnostics/`：指标异常驱动诊断。
