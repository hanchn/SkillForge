# 虚拟模特内容制作专家专业检查表

## 执行前

- [ ] 目标、范围、国家/渠道/系统、周期和责任人明确
- [ ] 已区分事实、假设、缺失数据和待确认口径
- [ ] 已读取一手资料，并记录来源时间与适用版本

## 核心维度

- [ ] 商品版型、纹理、颜色和遮挡 已定义、验证或标记为未知
- [ ] 人物身份、年龄表达、体型和肤色覆盖 已定义、验证或标记为未知
- [ ] 姿态、手部、接触点和物理合理性 已定义、验证或标记为未知
- [ ] 输入人像、肖像权和训练使用授权 已定义、验证或标记为未知
- [ ] 跨版本一致性、合成披露和失败样本 已定义、验证或标记为未知


## 失败模式

- 只有最佳实践，没有结合当前证据和约束。
- 只覆盖顺利路径，没有异常、补偿、回滚和人工介入。
- 指标同名但 scope、状态、时区、币种或归因窗口不同。
- 动作没有 owner、投入、依赖、验收或停止条件。

## 交付前

- [ ] 关键结论能追溯到证据或计算
- [ ] 备选方案、取舍和不做事项清楚
- [ ] 计划可分阶段执行并有验证与恢复路径
