# UGC 与达人内容制作

## 分类定位

管理创作者、寄样、brief、拍摄反馈、授权和交付。

## 选择原则

- 主角色负责需求、产能、预算、质量和交付，专项 Skill 负责策略、拍摄、AI 制作、后期或资产治理。
- 所有素材必须锁定商品事实、使用范围、授权、版本和质量门禁。
- AI 生成、虚拟模特和智能剪辑必须保留模型与修改血缘，并经过人工真实性和合规复核。

## 当前 Skill

- [UGC 与达人内容制作经理](ugc-content-production-manager/README.md)：管理创作者匹配、内容 brief、寄样、脚本边界、拍摄反馈、授权和交付，生产可信的原生用户内容。适用于UGC 广告素材采购、达人寄样内容制作、多创作者多角度内容测试，交付创作者画像与筛选表、UGC brief、脚本边界和反馈流程、授权、交付、版本和表现复盘。
