# Acceptance

- [ ] 所有供应商是否使用同一规格和报价口径
- [ ] 关键声明是否有可核验证据
- [ ] 样品、黄金样和量产变更是否受控
- [ ] 缺陷分级、检验和索赔责任是否明确
- [ ] 未发生：不得把供应商自述、证书截图或平台等级当作已验证事实
- [ ] 未发生：不得只按最低单价排序
- [ ] 未发生：未经样品和生产证据不得建议大货
- [ ] 未发生：规格、缺陷和验收必须可测量，避免好、优质、差不多等模糊词
- [ ] 事实、估算、假设和建议明确分开。
- [ ] 输出包含决策阈值和下一复核点。


## 交付物专项验收

- [ ] 已询问使用者现有数据、系统、模板、词库、规则、优先级和不可改变项
- [ ] 已标明时效事实的数据截止、采集时间、来源、版本和适用范围
- [ ] `RFQ 与供应商对比表` 已完整交付，并有下游使用者、验收证据和剩余风险
- [ ] `样品测试与质量控制计划` 已完整交付，并有下游使用者、验收证据和剩余风险
- [ ] `商业和供应风险清单` 已完整交付，并有下游使用者、验收证据和剩余风险
- [ ] `试单/量产建议` 已完整交付，并有下游使用者、验收证据和剩余风险
- [ ] 已覆盖正常路径、关键异常、暂停条件、人工升级、补偿或回滚
- [ ] 最终结论明确事实、假设、未知项、owner、审批和下一次复核时间
