# 市场采购 Skill 地图

## 一级定位

市场采购负责从市场需求和商品机会出发，完成选品、样品验证、供应商准入、目标成本、商务谈判、采购订单、交期质量和供应风险治理。采购不是单纯下单，而是以真实需求、商品差异、单位经济和供应可行为阶段门的投资决策。

## 主角色

- `选品与商品资深经理`：负责市场机会、选品组合、商品生命周期、目标成本和上市准备。
- `市场采购资深经理`：负责供应商、样品、成本谈判、采购订单、质量、交期和采购风险。
- 两个主角色均有专属周报 Skill。

## 专项能力

| 环节 | Skill | 核心职责 |
|---|---|---|
| 市场机会 | `market-demand-opportunity-analyst` | 人群需求、竞品、价格带、利润、供应可行和反证 |
| 选品验证 | `ecommerce-product-research` | 跨境商品机会研究、评分、验证、试点和退出 |
| 样品 | `sample-evaluation-sourcing-manager` | 打样、比样、测试、问题关闭、封样和量产放行 |
| 供应商质量 | `supplier-sourcing-quality-manager` | 供应商发现、准入、能力、质量、绩效和替代 |
| 商务谈判 | `supplier-negotiation-cost-manager` | 成本拆解、报价、MOQ、阶梯价、付款、交期和换项 |
| 采购执行 | `purchase-order-delivery-manager` | 订单、生产、质检、装运、付款证据、变更和关闭 |
| 采购风险 | `procurement-risk-compliance-manager` | 合规、产地、材料、集中度、产能和连续性 |

## 跨部门边界

- 渠道运营提供渠道机会、目标价格、销售节奏和上市需求，但不直接拥有选品和采购执行。
- 仓储库存负责补货计划、入库后的库存状态和仓配作业；市场采购负责采购供应与交期承诺。
- 法律政务负责合同、制裁、进出口和司法辖区法律判断；市场采购负责收集证据、执行控制和升级。
