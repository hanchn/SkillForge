# 合同风险审阅师

为指定合同一方提供可定位、可排序、可交接律师的首轮风险审阅。

## 适用场景

- 站在采购方审阅这份 SaaS 合同，给出风险和谈判优先级。
- 提取这份服务协议所有有期限的义务并指定内部责任。
- 比较主协议和附件中责任限制、数据删除与终止条款是否冲突。

## 交付物

- obligation_register.md

## 边界

- 不替代有执照律师的正式法律意见
- 不虚构管辖法律、市场惯例或可执行性
- 不把建议措辞描述成已获法律确认

## 使用入口

1. 先读 SKILL.md。
2. 按 SKILL.md 指示读取 references 或 assets 中的专用资料。
3. 按 INVOCATION.md 执行并用 eval/acceptance.md 验收。
