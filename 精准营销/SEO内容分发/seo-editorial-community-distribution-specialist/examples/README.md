# SEO 软文与社区内容分发专家场景示例

## 场景 1：SEO 软文和客座文章生成

- 调用目标：使用 `seo-editorial-community-distribution-specialist` 完成该场景的完整业务结果，不只给原则。
- 必须提供：目标、范围、真实现状证据、owner、时间、权限和已知约束。
- 需要追问：现有数据/系统/模板/词库/规则、使用者优先级、不可改变项、下游接收方式和验收人。
- 预期交付：关键词与选题证据包、渠道适配软文、帖子和回复草稿、发布矩阵、自检、互动和效果复盘。
- 故障注入：至少测试缺数据、口径冲突、权限不足或依赖失败中的一种，并给出暂停、补证、转人工或恢复路径。
