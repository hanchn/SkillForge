# 精准营销主角色

## 分类定位

沉淀品牌内容和增长营销主角色 Skill，并由主角色编排各专项执行 Skill。

## 选择原则

- 主角色 Skill 负责目标、取舍、协同和验收，专项 Skill 负责深度执行。
- 所有增长动作同时受利润、库存、合规、履约和客户体验约束。
- 数字结论必须组合数据看板或财务口径验证。

## 当前 Skill

- [品牌与内容高级经理](品牌与内容高级经理/senior-brand-content-manager/README.md)：统筹品牌定位、产品证据、内容体系、创意资产、社媒表达和多市场一致性。适用于由品牌与内容高级经理接管复杂经营任务、制定季度或月度计划、跨专项协同、经营诊断和复盘，交付角色经营目标与责任边界、专项 Skill 编排与决策、经营节奏、风险和团队行动计划。
- [品牌与内容高级经理周报](品牌与内容高级经理/senior-brand-content-manager-weekly-report/README.md)：基于真实经营证据生成品牌与内容高级经理视角的结论先行周报，并跟踪目标、驱动、风险、决策和责任闭环。适用于品牌与内容高级经理本周经营复盘、本周与上周或目标对比、管理层同步和跨团队行动跟踪，交付角色经营周报、指标变化和驱动解释、风险、决策与下周行动表。
- [增长营销高级经理](增长营销高级经理/senior-growth-marketing-manager/README.md)：统筹付费、自然、达人、邮件和活动增长，对获客质量、转化、留存和贡献利润负责。适用于由增长营销高级经理接管复杂经营任务、制定季度或月度计划、跨专项协同、经营诊断和复盘，交付角色经营目标与责任边界、专项 Skill 编排与决策、经营节奏、风险和团队行动计划。
- [增长营销高级经理周报](增长营销高级经理/senior-growth-marketing-manager-weekly-report/README.md)：基于真实经营证据生成增长营销高级经理视角的结论先行周报，并跟踪目标、驱动、风险、决策和责任闭环。适用于增长营销高级经理本周经营复盘、本周与上周或目标对比、管理层同步和跨团队行动跟踪，交付角色经营周报、指标变化和驱动解释、风险、决策与下周行动表。
