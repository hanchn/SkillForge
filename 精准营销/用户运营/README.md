# 用户运营

## 分类定位

覆盖邮件生命周期、购后、召回、会员和忠诚度。

## 选择原则

- 主角色 Skill 负责目标、取舍、协同和验收，专项 Skill 负责深度执行。
- 所有增长动作同时受利润、库存、合规、履约和客户体验约束。
- 数字结论必须组合数据看板或财务口径验证。

## 当前 Skill

- [用户生命周期、会员与挽留经理](customer-retention-loyalty-manager/README.md)：基于统一客户身份、同意、客户价值和行为周期，设计新客培育、老客复购、会员权益、流失预警、召回挽留和忠诚增长闭环。适用于新客首购后培育与二购提升、新老客分层、生命周期旅程和老客经营、会员、等级、积分、权益和推荐计划、沉默、流失预警、召回与挽留、留存下降、会员成本或复购质量诊断，交付新客、老客、会员和流失定义及数据质量报告、生命周期分层、状态机、触发、频控和跨渠道旅程、会员权益、积分负债、成本、毛利和体验模型、召回挽留实验、增量评估、止损与复盘。
- [邮件生命周期营销师](email-lifecycle-marketer/README.md)：按同意、用户阶段和行为信号设计可衡量的邮件与自动化流程。适用于欢迎、弃购、购后、召回流程、Newsletter 运营、邮件收入和送达率治理，交付生命周期流程图、分群、触发与内容方案、实验和送达健康看板。
