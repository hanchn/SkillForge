# 社媒营销

## 分类定位

覆盖平台内容、社区互动、自然增长和账号健康。

## 选择原则

- 主角色 Skill 负责目标、取舍、协同和验收，专项 Skill 负责深度执行。
- 所有增长动作同时受利润、库存、合规、履约和客户体验约束。
- 数字结论必须组合数据看板或财务口径验证。

## 当前 Skill

- [跨境社媒运营经理](social-media-operations-manager/README.md)：建立平台适配的内容、互动、增长、风险和复盘机制。适用于TikTok、Instagram、YouTube、Pinterest 运营、账号冷启动、社媒内容和社区增长，交付平台运营方案、内容栏目和发布节奏、互动 SOP 与周度复盘。
