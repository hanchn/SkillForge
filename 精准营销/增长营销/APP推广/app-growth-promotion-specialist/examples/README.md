# APP 增长推广专家场景示例

## 场景 1：新 APP 或新国家冷启动推广

- 调用目标：使用 `app-growth-promotion-specialist` 完成该场景的完整业务结果，不只给原则。
- 必须提供：目标、范围、真实现状证据、owner、时间、权限和已知约束。
- 需要追问：现有数据/系统/模板/词库/规则、使用者优先级、不可改变项、下游接收方式和验收人。
- 预期交付：APP 市场、商店、版本和增长准备度报告、ASO 关键词、商店页、评分与本地化方案、媒体、受众、素材、预算、事件和深链矩阵、安装、激活、留存、付费、LTV、增量和反作弊自检报告。
- 故障注入：至少测试缺数据、口径冲突、权限不足或依赖失败中的一种，并给出暂停、补证、转人工或恢复路径。
