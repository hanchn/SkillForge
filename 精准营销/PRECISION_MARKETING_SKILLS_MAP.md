# 精准营销 Skill 地图

## 一级定位

精准营销负责“把正确的信息在正确时间触达正确人群，并证明增量价值”，覆盖市场策略、品牌、内容、SEO/GEO、广告、社媒、达人、创意、活动、邮件、留存、归因和增长实验。

## 主角色

- 品牌与内容资深经理：负责品牌定位、内容资产、跨市场表达和自然影响力。
- 增长营销资深经理：负责渠道预算、获客质量、转化、留存、LTV 和增量验证。
- 用户运营资深经理：负责用户分层、生命周期、邮件、购后、会员、召回和客户价值。

两个角色目录均包含主角色 Skill 和专属周报 Skill。

## 专项能力链

| 环节 | 专项能力 |
|---|---|
| 市场与人群 | 市场情报、竞争洞察、受众、定位和上市策略 |
| 内容与发现 | 内容营销、SEO、GEO、社媒和多语言表达 |
| 拉新与首购 | 新客定义、渠道组合、创意落地页、CAC、回收期和增量验证 |
| 创意与触达 | 定义人群、信息、渠道和效果需求，并向一级 `创拍视觉` 职能提交制作 brief；本域保留广告、达人联盟、活动和邮件自动化 |
| 转化与留存 | 落地页协同、客户生命周期、忠诚度和召回 |
| 衡量与增长 | UTM、归因、增量实验、增长模型和预算优化 |

## 与渠道运营的接口

- 精准营销不得脱离真实商品事实、库存、价格、合规和履约能力制造需求。
- 渠道运营不得只看平台 GMV 判断营销增量。
- 双方以统一指标、贡献利润、客户质量和可持续复购共同验收。

## 与创拍视觉的接口

- 精准营销负责受众、定位、传播任务、渠道、预算和效果反馈。
- 创拍视觉负责策略落地、拍摄或生成、制作质量、真实性、授权、版本与交付。
- 素材表现回流精准营销，但不能脱离受众、投放和版位条件直接评价制作质量。
